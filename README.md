# AdamShell / Universe

**AdamShell / Universe** je symbolický Python projekt, který modeluje vznik a vývoj vlastního programovatelného světa.

Projekt není jen obyčejný skript. Je to základ pro interaktivní prostředí, ve kterém se postupně skládají vrstvy reality: fyzika, čas, prostor, kvantová neurčitost, božské entity, kočky, knihovna, místo setkání, Eden, historie a později i zjevení.

Cílem projektu je vytvořit systém, ve kterém se mytologické, filozofické a programátorské prvky chovají jako části jednoho živého světa.

## Základní myšlenka

Universe začíná jako prázdná kořenová realita.

Postupně se v ní aktivují vrstvy:

- fyzika
- čas
- prostor
- energie
- kvantová vrstva
- božská vrstva
- idea entities
- cats layer
- library
- meeting place
- Eden
- history layer
- revelation layer

Každá vrstva má vlastní odpovědnost a vlastní místo v příběhu systému.

Projekt se dá chápat jako kombinace symbolického operačního systému, textové hry, mytologického sandboxu, filozofického modelu reality a experimentálního Python enginu.

## Spuštění projektu

Nejdřív aktivuj virtuální prostředí:

```powershell
.\.venv\Scripts\Activate.ps1
```

Potom spusť hlavní soubor:

```powershell
python main.py
```

## Vývojová filozofie

Projekt se staví postupně.

Nejdřív vzniká svět.  
Potom vznikají vrstvy.  
Potom vznikají entity.  
Potom vzniká příběh.  
A nakonec z toho může vzniknout interaktivní prostředí.

Universe tedy není jen kolekce Python souborů.  
Je to pokus napsat svět jako program.
